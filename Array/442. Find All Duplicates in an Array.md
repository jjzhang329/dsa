# 442. Find All Duplicates in an Array
[leetcode](https://leetcode.com/problems/find-all-duplicates-in-an-array/description/)

```python
class Solution:
    def findDuplicates(self, nums: List[int]) -> List[int]:
        dups = set()
        result = []
        for num in nums:
            if num in dups:
                result.append(num)
            else:
                dups.add(num)
        
        return result
```
